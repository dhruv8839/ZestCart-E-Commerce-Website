# 🛒 ZestCart – Modern E-Commerce Web Application

ZestCart is a modern, responsive e-commerce web application built to deliver a smooth online shopping experience. It features product browsing, smart search, filtering, wishlist, shopping cart, checkout, and a clean user interface with dark/light mode support.

---

## ✨ Features

- 🏠 Modern Home Page
- 🛍️ Product Catalog
- 🔍 Product Search
- 🗂️ Category Filtering
- 💰 Price Filtering
- 📊 Product Sorting
- ❤️ Wishlist
- 🛒 Shopping Cart
- ➕ Update Product Quantity
- 🧾 Checkout Flow
- 🎟️ Coupon Support
- 🌙 Dark & ☀️ Light Theme
- 📱 Fully Responsive Design
- 🔔 Toast Notifications
- ⚡ Fast & Modern UI
- 🎨 Smooth Animations

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/dhruv8839/ZestCart-E-Commerce-Website.git
```

Go to project

```bash
cd ZestCart-E-Commerce-Website
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
